# App Álcool ou Gasolina

## 📱 Sobre o projeto

O **App Álcool ou Gasolina** é um aplicativo Android simples desenvolvido com o objetivo de ajudar o usuário a decidir qual combustível compensa mais abastecer: **álcool (etanol)** ou **gasolina**, com base nos preços informados.

A regra utilizada é:

- Se o preço do álcool for menor que 70% do preço da gasolina → compensa usar álcool
- Caso contrário → compensa usar gasolina

Este projeto foi desenvolvido com fins de estudo e prática de desenvolvimento mobile utilizando Android.

---

## 🚀 Tecnologias utilizadas

- Java
- Android Studio
- XML
- ConstraintLayout

---

## 📸 Funcionalidades

- Inserção do preço do álcool
- Inserção do preço da gasolina
- Cálculo automático da melhor opção
- Exibição do resultado na tela
- Validação de campos

## 🛠️ Como executar o projeto

### Pré-requisitos

- Android Studio instalado
- JDK configurado
- Emulador Android ou dispositivo físico

### Passos

1. Clone este repositório:

```bash
git clone https://github.com/Tainan-Rocha/AppAlcool.git
````
2. Abra o projeto no Android Studio
3. Aguarde o Gradle finalizar a sincronização
4. Execute o projeto no emulador ou dispositivo
